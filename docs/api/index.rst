API Reference
=============

This section contains detailed API documentation for all modules in pyMyriad.

------------

Analysis Tree Construction
--------------------------

The :mod:`pyMyriad.analysis_tree` module provides classes for building hierarchical analysis trees.

Data Tree Results
-----------------

The :mod:`pyMyriad.data_tree` module contains classes that hold the results of executing an analysis tree.

Data Processing
---------------

The :mod:`pyMyriad.tabular` module provides functions for flattening and formatting data trees.

Visualization
-------------

The :mod:`pyMyriad.plots` module offers plotting functions for visualizing analysis results.

Table Generation
----------------

The :mod:`pyMyriad.listing` module contains functions for generating formatted tables.

Utilities
---------

The :mod:`pyMyriad.utils` module provides helper functions for expression evaluation and string conversion.
