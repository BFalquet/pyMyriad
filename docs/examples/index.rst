Example Notebooks
=================

Learn pyMyriad through interactive Jupyter notebooks.

.. note::
   These notebooks are not executed during the documentation build.
   Download them and run locally for an interactive experience.

Notebook Descriptions
---------------------

1. **Getting Started** - Introduction to pyMyriad basics
   
   * Installing pyMyriad
   * Creating your first AnalysisTree
   * Running analyses on data

2. **Analysis Trees** - Deep dive into tree construction
   
   * Understanding splits and analyses
   * Path-based operations
   * Termination signals

3. **Tables and Listings** - Exporting results to tables
   
   * Creating simple tables
   * Pivot tables
   * Cascade tables

4. **Plots** - Visualizing your results
   
   * Forest plots
   * Distribution plots
   * Customizing visualizations

5. **Formatting Statistics** - Formatting output
   
   * Custom format strings
   * Statistical formatting
   * Great Tables integration

6. **Advanced Topics** - Advanced features
   
   * Cross-group analysis
   * Custom environments
   * Performance optimization

Download Notebooks
------------------

All notebooks are available in the ``examples/notebooks/`` directory of the repository.

Clone the repository:

.. code-block:: bash

   git clone https://github.com/yourusername/pyMyriad.git
   cd pyMyriad/examples/notebooks
   jupyter notebook
